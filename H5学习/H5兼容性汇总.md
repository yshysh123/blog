# 在 iOS 中，输入框弹起键盘前后，页面都可能出现问题，需要监听下键盘弹起收起的状态，然后自己滚动

```javascript
document.body.addEventListener('focusout', () => {
  if (isIphone()) {
    setTimeout(() => {
      document.body.scrollTop = document.body.scrollHeight
    }, 100)
  }
})

document.body.addEventListener('focusin', () => {
  if (isIphone()) {
    setTimeout(() => {
      document.body.scrollTop = document.body.scrollHeight
    }, 100)
  }
})
```
